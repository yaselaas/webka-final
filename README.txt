OtakuPins - simple student project (Node + MongoDB + external Anime API)

How to run:
1) Install dependencies:
   npm i

2) Create .env (copy from .env.example) and set MONGO_URI and JWT_SECRET

3) Start server:
   npm start

Open:
http://localhost:3000

Notes:
- Put your own anime images into: client/assets/images/
- Anime search works without API key (Jikan fallback).
